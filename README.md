# 🎯 DigiTools - Premium Digital Marketplace

DigiTools is a modern e-commerce platform designed for tech enthusiasts to browse and purchase high-quality digital assets, software subscriptions, and developer tools.

## 🌐 Live Site
https://b13-a6-digi-tools-platform-liart.vercel.app/

## 🚀 Features
* **Dynamic Product Toggling:** Seamlessly switch between the Product Gallery and your Shopping Cart without page reloads.
* **Interactive Cart System:** Real-time cart updates with the ability to add, remove, and bulk-checkout using `react-toastify` for instant feedback.
* **Fully Responsive UI:** A "pixel-perfect" design built with Tailwind CSS and DaisyUI, optimized for mobile, tablet, and desktop views.

## ⚙️ Technologies Used
* **React.js** (Functional Components, Hooks)
* **Tailwind CSS & DaisyUI** (For styling and UI components)
* **JavaScript (ES6+)**
* **React-Toastify** (User notifications)
* **JSON** (Local data management)
